# U9 - Arreglos

**Arreglo**: Conj. de valores en el mismo sistema numerico almacenados en un bloque.

Mediante repeticion se pueden recorrer. Ejemplo:

```js
recorrido: MOV R1, 0x0000  // Inicia el contador

ciclo: CMP R1, 0x0004
       JG fin
       ADD R1, 0x0001
       JMP ciclo

fin: RET
```
